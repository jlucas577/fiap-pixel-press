Objetivo: Criar a arquitetura de um sistema (projeto livre) para ser a base de implementação da próxima aula.

Esta atividade vale 1/4 da nota da disciplina.

    Defina (pelo menos):
        Funcionalidades principais
        Tipos de usuários e suas permissões
        Diagrama de arquitetura (camadas: frontend, backend, banco de dados)
        Entidades principais e relacionamentos
        Endpoints da API (principais rotas REST)
        Tecnologias sugeridas (pode ser genérico: "banco relacional", "framework web", etc.)
        Fluxos principais (diagrama ou descrição textual)
    Entregável:
        Documento PDF ou Markdown com diagramas (pode usar draw.io, Excalidraw, Mermaid, etc.).
        Prompts utilizados para criar a arquitetura, indicando a ferramenta utilizada (ex: ChatGPT para as classes + Mermaid para os diagramas, etc.).

Obs.: tente trazer sua experiência profissional para o projeto. Especifique bem a arquitetura desejada, as tecnologias a serem usadas e regras de negócio importantes. Deixe isso claro no prompt, pois será o foco maior da avaliação desta atividade.
